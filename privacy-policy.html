Privacy Policy for Google Enhanced

Effective date: March 11, 2026

Google Enhanced is a Chrome extension that removes Google homepage doodles and share buttons, replaces the Chrome new tab page with a customizable page, and provides related interface controls. This policy explains what data the extension handles, how it is used, and whether it is shared.

What data the extension handles

  • Chrome profile email address, if available through the chrome.identity API, to show the signed-in account chip on the custom new tab page.
  • Most visited sites from the chrome.topSites API to create initial shortcut tiles on the custom new tab page.
  • User-provided content and settings stored locally, such as custom shortcuts, uploaded background images, custom avatar images, doodle preferences, theme choices, and UI state.
  • Limited page context needed to apply the extension's UI changes on supported Google and YouTube pages. The extension modifies page appearance locally in the browser and does not send page content to the developer.

How the data is used

  • To display the signed-in account chip and avatar options on the custom new tab page.
  • To show initial shortcut suggestions based on the browser's most visited sites.
  • To save user customization choices and restore them across browser sessions.
  • To enable the extension's homepage and new tab interface features.

Where data is stored

Settings and user-provided content are stored locally in the browser using chrome.storage.local and browser local storage. Google Enhanced does not send this data to the developer's own servers.

Data sharing

Google Enhanced does not sell user data and does not share user data with data brokers, advertisers, or analytics providers.

Some extension features interact directly with Google-owned services from the user's browser. For example, the account chip and Google services UI may request resources from Google domains. When a user opens a search, image search, Google app, or shortcut destination, the browser connects directly to the relevant site selected by the user.

Data retention

Locally stored settings and user-provided content remain on the device until the user changes them, clears extension storage, resets the browser profile, or uninstalls the extension.

User controls

  • Users can edit or remove shortcuts at any time.
  • Users can replace or clear custom backgrounds and custom avatar images at any time.
  • Users can sign out of the local account chip display from the extension UI.
  • Users can remove the extension to delete its locally stored data from Chrome.

Children's privacy

Google Enhanced is not directed to children and is not intended for use by children under 13.

Chrome Web Store Limited Use statement

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Contact
